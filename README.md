# Vue ID Card Generator

## Quick Start
1. Unzip folder
2. Rename your template image (MOCKUP-1.jpg) to `public/template.png` and replace the placeholder
3. Open terminal: `npm install`
4. `npm run dev` - opens http://localhost:3000

## Features
- Drag/drop profile pic
- Live drag/resize/rotate/flip
- Text overlays (name/region left upper)
- Download exact PNG preview

Adjust positions in App.vue computed configs to fit your template perfectly.

Your template.png is ready - just replace placeholder!
